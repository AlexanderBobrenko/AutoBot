## PROMPT #1 - БАЗОВЫЙ БОТ (MVP)
**Статус:** ⏳ Ожидает выполнения
**Время выполнения:** ~10 минут
Ты senior Python разработчик. Напиши Telegram бота для учета расходов на автомобиль.

Стек: Python 3.11+, aiogram 3.x, SQLite3

Требования:

Команда /start приветствует пользователя и показывает ReplyKeyboardMarkup с кнопками:

"➕ Добавить расход"

"📊 Статистика за сегодня"

"📈 Статистика за неделю"

Функционал "Добавить расход":

Запрашивает сумму (только цифры)

Запрашивает категорию (InlineKeyboardMarkup: Топливо, Ремонт, Мойка, Страховка, Другое)

Запрашивает комментарий (можно пропустить, отправив "Пропустить")

Сохраняет в SQLite: user_id, amount, category, comment, date (автоматически)

Статистика за сегодня: выводит сумму расходов и список по категориям

Статистика за неделю: выводит общую сумму и расходы по дням

Структура файлов:

config.py: хранение TOKEN через os.getenv()

database.py: класс Database с методами init_db(), add_expense(), get_today_stats(), get_week_stats()

keyboards.py: функции get_main_keyboard(), get_categories_keyboard()

bot.py: все хендлеры и запуск

Напиши полный код каждого файла. Добавь обработку ошибок и логирование через print().
